# 30 – Benutzereingaben

## Themenübersicht

- Eingaben von Benutzern mit `prompt()` entgegennehmen
- Umwandlung von Strings in Zahlen
- Verwendung von `Number()` und `parseInt()` zur Typkonvertierung
- Verarbeitung von Benutzereingaben in Programmen
- Kombination von Eingaben mit Kontrollstrukturen

## Lernziele nach Bloom-Taxonomie

| Nr. | Lernziel                                                                             | Beschreibung                                                                                | Taxonomiestufe |
| --- | ------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------- | -------------- |
| 1   | Die Schüler:innen können die Funktion `prompt()` benennen.                           | Angeben, welche Funktion für Benutzereingaben verwendet wird.                               | K1 – Wissen    |
| 2   | Die Schüler:innen können den Unterschied zwischen String- und Number-Typen erklären. | Verstehen, warum `prompt()` immer einen String zurückgibt und wann Konvertierung nötig ist. | K2 – Verstehen |
| 3   | Die Schüler:innen können Benutzereingaben mit `prompt()` entgegennehmen.             | String-Eingaben vom Benutzer abfragen und in Variablen speichern.                           | K3 – Anwenden  |
| 4   | Die Schüler:innen können Strings in Zahlen konvertieren.                             | `Number()` und `parseInt()` zur Typumwandlung nutzen.                                       | K3 – Anwenden  |
| 5   | Die Schüler:innen können Benutzereingaben validieren.                                | Eingaben prüfen und mit if-Bedingungen auf Gültigkeit testen.                               | K3 – Anwenden  |
