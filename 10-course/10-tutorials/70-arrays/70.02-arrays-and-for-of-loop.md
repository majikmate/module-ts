# Arrays & for of loop

```typescript
// calculate the sum of the values 1, 2, 3, 4

const numbers = [1, 2, 3, 4];

let sumOfNumbers = 0;

for (const num of numbers) {
    sumOfNumbers = sumOfNumbers + num;
}

console.log(sumOfNumbers);
```
