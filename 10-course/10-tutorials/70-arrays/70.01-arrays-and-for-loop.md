# Arrays & for loop

```typescript
// calculate the sum of the values 1, 2, 3, 4

const numbers = [1, 2, 3, 4];

let sumOfNumbers = 0;

for (let i = 0; i < numbers.length; i = i + 1) {
    sumOfNumbers = sumOfNumbers + numbers[i];
}

console.log(sumOfNumbers);
```
