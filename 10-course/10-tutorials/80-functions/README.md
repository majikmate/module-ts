# 80 – Funktionen

## Themenübersicht

- Funktionsdefinition und -aufruf
- Parameter und Rückgabewerte
- Funktionen als Werte (First-Class Functions)
- Zuweisung von Funktionen an Variablen
- Arrow-Funktionen (Pfeilfunktionen)
- Unterschiede zwischen traditionellen und Arrow-Funktionen

## Lernziele nach Bloom-Taxonomie

| Nr. | Lernziel                                                               | Beschreibung                                                                        | Taxonomiestufe |
| --- | ---------------------------------------------------------------------- | ----------------------------------------------------------------------------------- | -------------- |
| 1   | Die Schüler:innen können die Syntax von Funktionen benennen.           | Funktionsdeklaration, Parameter, Rückgabewerte und return-Statement identifizieren. | K1 – Wissen    |
| 2   | Die Schüler:innen können erklären, was First-Class Functions bedeutet. | Verstehen, dass Funktionen wie Werte behandelt werden können.                       | K2 – Verstehen |
| 3   | Die Schüler:innen können Funktionen definieren und aufrufen.           | Eigene Funktionen mit Parametern erstellen und verwenden.                           | K3 – Anwenden  |
| 4   | Die Schüler:innen können Funktionen Variablen zuweisen.                | Funktionen als Werte speichern und über Variablen aufrufen.                         | K3 – Anwenden  |
| 5   | Die Schüler:innen können Arrow-Funktionen verwenden.                   | Die kompakte Syntax von Pfeilfunktionen anwenden.                                   | K3 – Anwenden  |
