# 90 – Array-Iterator-Methoden

## Themenübersicht

- Höhere Funktionen (Higher-Order Functions) für Arrays
- `forEach()` - Iteration über Array-Elemente
- `find()` - Suche nach einem Element
- `some()` - Prüfung auf mindestens ein passendes Element
- `every()` - Prüfung ob alle Elemente eine Bedingung erfüllen
- `filter()` - Filtern von Array-Elementen
- `map()` - Transformation von Array-Elementen
