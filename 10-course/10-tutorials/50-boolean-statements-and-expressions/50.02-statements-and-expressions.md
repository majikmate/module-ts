# Statements & Expressions

```typescript
// statement: define the length of a rectangle
let a = 7;
let b = 9;

// expression: calculate the area of the rectangle with the expression: a * b
let area = a * b;
```

**Statements**

- Statements beginnen immer am Anfang und enden mit einem Strichpunkt, z.B.:
  - `let a = 7;`
  - `console.log("Olaf Gustavson";`
- Statements können auch Kontrollstrukturen sein, z.B. das if Statement:
  ```typescript
  if (a < 10) {
      console.log("Der Wert der Variable a ist kleiner 10");
  }
  ```

**Expressions**

- Expressions liefern immer einen Wert zurück
- Expressions können unter anderem ...
  - einfache Zahlen oder Zeichenketten (strings) sein, z.B. `7` oder
    `"Olaf Gustavson"`
  - mathematische Operationen sein, z.B. `7 + 9` oder `a * b`
  - Zeichen-Verkettungen (string concatenation) sein, z.B.
    `firstname + " " + lastName`
  - Vergleiche sein, z.B. `a <= 10` oder `a > b`
  - Boolesche Operationen sein, z.B. `a > 10 && a <= b`
- Etwas in einer runden Klammer ist immer eine Expression, z.B.
  - `(a <= b)`
  - `(a + b) * c`
  - `console.log("Olaf Gustavson");`
