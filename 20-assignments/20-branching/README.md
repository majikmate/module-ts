# Verzweigungen - Aufgaben

Dieses Kapitel enthält Übungsaufgaben zum Thema if-Anweisungen und
Kontrollstrukturen in TypeScript.

## Übersicht

- [20.01 If-Kontrollblock 01](20.01-if-control-block-01/README.md) - Einfache
  if-Anweisungen
- [20.02 If-Kontrollblock 02](20.02-if-control-block-02/README.md) -
  If-else-Anweisungen

## Lernziele

Nach Abschluss dieser Aufgaben kannst du:

- Einfache `if`-Anweisungen schreiben
- `if-else`-Anweisungen verwenden
- `if-else-if`-Ketten erstellen
- Vergleichsoperatoren verwenden (`>`, `<`, `>=`, `<=`, `===`, `!==`)
- Scopes (Gültigkeitsbereiche) verstehen
- Verschachtelte if-Anweisungen schreiben

## Verwendung

Jede Aufgabe enthält:

1. Eine `README.md` Datei mit den Anweisungen
2. Ein `workspace/main.ts` Datei zum Implementieren deiner Lösung
