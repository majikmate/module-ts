# Positive Zahlen

Schreibe ein kleines Programm, das prüft, ob eine Zahl positiv, negativ oder
null ist.

## Anweisungen

- Erstelle eine Variable `number` und weise ihr einen Wert zu.
- Verwende `if`-Anweisungen, um zu prüfen:
  - Wenn die Zahl größer als 0 ist
    - gib `Die Zahl ist positiv` aus
  - Wenn die Zahl kleiner als 0 ist
    - gib `Die Zahl ist negativ` aus
  - Wenn die Zahl gleich 0 ist
    - gib `Die Zahl ist null` aus

**Hinweis: Verwende nur `if`, `<`, `>` und `==`.**

## Abgabe

Bitte gib deine Arbeit in der Datei `main.ts` im Ordner `workspace` ab:
