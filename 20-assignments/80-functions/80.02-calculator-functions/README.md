# Taschenrechner-Funktionen

Erstelle einen Taschenrechner mit verschiedenen Funktionen für mathematische
Operationen.

## Anweisungen

### 1. Aufgabe

Schreibe folgende Funktionen:

#### a. `add(a: number, b: number): number`

- Addiert zwei Zahlen und gibt das Ergebnis zurück

#### b. `subtract(a: number, b: number): number`

- Subtrahiert die zweite Zahl von der ersten und gibt das Ergebnis zurück

#### c. `multiply(a: number, b: number): number`

- Multipliziert zwei Zahlen und gibt das Ergebnis zurück

#### d. `divide(a: number, b: number): number`

- Dividiert die erste Zahl durch die zweite und gibt das Ergebnis zurück

#### e. `power(base: number, exponent: number): number`

- Berechnet base^exponent (z.B. 2^3 = 8)
- Verwende eine Schleife oder den `**` Operator

### 2. Verwendung

- Rufe jede Funktion mit verschiedenen Werten auf
- Speichere die Ergebnisse in Variablen
- Gib die Ergebnisse mit aussagekräftigen Beschreibungen aus

### 3. Hinweise

- Alle Funktionen geben einen Wert zurück (`: number`)
- Verwende `return` um das Ergebnis zurückzugeben
- Teste auch Randfälle (z.B. Division durch 0 gibt `Infinity`)

### 4. Erwartete Ausgabe

```
5 + 3 = 8
10 - 4 = 6
6 * 7 = 42
20 / 4 = 5
2^3 = 8
```
