# Funktionen - Aufgaben

Dieses Kapitel enthält Übungsaufgaben zum Thema Funktionen in TypeScript.

## Übersicht

### Standard-Funktionen (80.01 - 80.05)

- [80.01 Begrüßungs-Funktionen](./80.01-greeting-functions/README.md) -
  Funktionen mit Parametern und Rückgabewerten
- [80.02 Taschenrechner-Funktionen](./80.02-calculator-functions/README.md) -
  Mathematische Operationen als Funktionen
- [80.03 Temperatur-Umrechner](./80.03-temperature-converter/README.md) -
  Funktionen für Temperaturumrechnungen
- [80.04 Zahlen-Validator](./80.04-number-validator/README.md) -
  Validierungsfunktionen für Zahlen
- [80.05 String-Operationen](./80.05-string-operations/README.md) - Funktionen
  für String-Manipulation

### Arrow-Funktionen (80.06 - 80.10)

- [80.06 Arrow-Funktionen: Grundlagen](./80.06-arrow-basics/README.md) -
  Einführung in Arrow-Funktionen
- [80.07 Array-Transformationen](./80.07-arrow-transformations/README.md) -
  Arrays mit Arrow-Funktionen transformieren
- [80.08 Funktionen als Werte](./80.08-functions-as-values/README.md) -

- [80.09 Callback-Funktionen](./80.09-callback-functions/README.md) - Callbacks
  mit Arrow-Funktionen
- [80.10 Funktions-Vergleich](./80.10-function-comparison/README.md) -
  Traditionelle vs. Arrow-Funktionen

## Lernziele

Nach Abschluss dieser Aufgaben kannst du:

- Funktionen mit Parametern und Rückgabewerten schreiben
- Zwischen `void`-Funktionen und Funktionen mit Return-Wert unterscheiden
- Arrow-Funktionen mit implizitem und explizitem Return verwenden
- Funktionen als Werte speichern und übergeben
- Callback-Funktionen implementieren
- Die Unterschiede zwischen traditionellen Funktionen und Arrow-Funktionen
  verstehen

## Verwendung

Jede Aufgabe enthält:

1. Eine `README.md` Datei mit den Anweisungen
2. Ein `workspace/main.ts` Datei zum Implementieren deiner Lösung
