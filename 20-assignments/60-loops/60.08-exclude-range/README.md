# Exclude Range

Schreibe ein Programm, das alle Zahlen von 1 bis 50 ausgibt, aber dabei die
Zahlen von 20 bis 30 überspringt.

## Anweisungen

### 1. Aufgabe

Verwende eine Schleife, um durch die Zahlen von 1 bis 50 zu iterieren:

- Gib alle Zahlen aus, AUSSER die Zahlen im Bereich von 20 bis 30 (inklusive)
- Verwende `if`-Anweisungen oder `continue`, um die Zahlen im Bereich 20-30 zu
  überspringen

### 2. Erwartete Ausgabe

Die Ausgabe sollte so aussehen:

```
1
2
3
...
19
31
32
...
50
```

## Abgabe

Bitte gib deine Arbeit in der Datei `main.ts` im Ordner `workspace` ab:

[workspace/main.ts](./workspace/main.ts)
