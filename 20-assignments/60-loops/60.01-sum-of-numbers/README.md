# Sum of Numbers 1–100

Schreibe ein Programm, das die Summe aller Zahlen von 1 bis 100 berechnet.

## Anweisungen

### 1. Aufgabe

- Verwende eine `for`-Schleife, um durch die Zahlen von 1 bis 100 zu iterieren
- Addiere jede Zahl zu einer Summenvariable
- Gib am Ende die Gesamtsumme aus

### 2. Hinweise

- Erstelle eine Variable für die Summe (z.B. `sum`) und initialisiere sie mit 0
- In der Schleife addierst du jede Zahl zur Summe: `sum = sum + i` oder
  `sum += i`
- Die erwartete Summe ist: 1 + 2 + 3 + ... + 99 + 100

### 3. Erwartete Ausgabe

```
5050
```
