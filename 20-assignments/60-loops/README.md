# Schleifen - Aufgaben

Dieses Kapitel enthält Übungsaufgaben zum Thema Schleifen in TypeScript.

## Übersicht

- [60.01 Summe von Zahlen](60.01-sum-of-numbers/README.md) - Zahlen summieren
- [60.02 Summe ungerader Zahlen](60.02-sum-of-odd-numbers/README.md) - Ungerade
  Zahlen summieren
- [60.03 Countdown](60.03-countdown/README.md) - Rückwärts zählen
- [60.04 Zahlen überspringen](60.04-skip-numbers/README.md) - Mit `continue`
  arbeiten
- [60.05 Summe von A bis B](60.05-sum-from-a-to-b/README.md) - Bereich summieren
- [60.06 Durchschnitt berechnen](60.06-calculate-average/README.md) - Mittelwert
  berechnen
- [60.07 Summe von Vielfachen](60.07-sum-of-multiples/README.md) -
  Modulo-Operator verwenden
- [60.08 Bereich ausschließen](60.08-exclude-range/README.md) - Bereiche
  überspringen

## Lernziele

Nach Abschluss dieser Aufgaben kannst du:

- `for`-Schleifen schreiben und verwenden
- Mit Schleifenvariablen arbeiten
- `continue` zum Überspringen verwenden
- `break` zum Abbrechen verwenden
- Berechnungen in Schleifen durchführen
- Den Modulo-Operator `%` für Vielfache nutzen
- Schleifen für verschiedene Bereiche und Schrittweiten einsetzen

## Verwendung

Jede Aufgabe enthält:

1. Eine `README.md` Datei mit den Anweisungen
2. Ein `workspace/main.ts` Datei zum Implementieren deiner Lösung
