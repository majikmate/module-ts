# Calculate Average

Schreibe ein Programm, das den Benutzer fragt, wie viele Zahlen er eingeben
möchte, dann diese Zahlen einliest und den Durchschnitt berechnet.

## Anweisungen

### 1. Aufgabe

- Frage den Benutzer, wie viele Zahlen er eingeben möchte
- Verwende eine Schleife, um die entsprechende Anzahl an Zahlen einzulesen
- Berechne den Durchschnitt aller eingegebenen Zahlen
- Gib den Durchschnitt aus

### 2. Hinweise

- 👉 **Hint:** Durchschnitt = Gesamtsumme ÷ Anzahl
- Erstelle Variablen für:
  - Die Anzahl der Zahlen (`count`)
  - Die Summe aller Zahlen (`sum`)
- Verwende `prompt()` zum Einlesen der Benutzereingaben
- Vergiss nicht, die Eingaben in Zahlen umzuwandeln

### 3. Beispiel

```
Wie viele Zahlen möchtest du eingeben? 3
Gib Zahl 1 ein: 10
Gib Zahl 2 ein: 20
Gib Zahl 3 ein: 30
Durchschnitt: 20
```
