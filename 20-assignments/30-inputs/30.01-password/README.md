# Password check

Schreibe ein Programm, dass überprüft, ob ein eingegebenes Passwort korrekt ist

## Anweisungen

- Deklariere die Variablen `input` und `password` und setze `password` auf das
  geheime Passwort
- Benutze `prompt` um ein Password über die Eingabe in der Kommandozeile in
  `input` zu speichern
- Benutze das if statement und prüfe:
  - Wenn `input` das gleiche enthält wie `password` dann gib `Access granted`
    aus
  - Wenn `input` das ungleich `password` dann gib `Access denied` aus

## Abgabe

Bitte gib deine Arbeit in der Datei `main.ts` im Ordner `workspace` ab:

[workspace/main.ts](./workspace/main.ts)
