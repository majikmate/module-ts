# Eingaben - Aufgaben

Dieses Kapitel enthält Übungsaufgaben zum Thema Benutzereingaben in TypeScript.

## Übersicht

- [30.01 Passwort](30.01-password/README.md) - Passwort-Prüfung
- [30.02 Zahlen-Prüfung](30.02-number-check/README.md) - Zahlen validieren
- [30.03 Taschenrechner](30.03-calculator/README.md) - Einfacher Rechner
- [30.04 Alters-Prüfung](30.04-age-check/README.md) - Alter überprüfen
- [30.05 Temperatur-Ratschläge](30.05-temperature-advice/README.md) -
  Wetterempfehlungen
- [30.06 Noten-Bewertung](30.06-grade-evaluation/README.md) - Noten
  klassifizieren
- [30.07 Getränke-Bestellung](30.07-drink-order/README.md) - Getränke auswählen

## Lernziele

Nach Abschluss dieser Aufgaben kannst du:

- Benutzereingaben mit `prompt()` einlesen
- Strings in Zahlen umwandeln mit `Number()`
- Eingaben validieren und prüfen
- If-Anweisungen mit Eingaben kombinieren
- Berechnungen mit Eingabewerten durchführen
- Mehrfache if-else-if Verzweigungen verwenden
- Benutzerfreundliche Programme schreiben

## Verwendung

Jede Aufgabe enthält:

1. Eine `README.md` Datei mit den Anweisungen
2. Ein `workspace/main.ts` Datei zum Implementieren deiner Lösung
