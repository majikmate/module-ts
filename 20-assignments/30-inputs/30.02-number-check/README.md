# Positive Zahlen mit Else

Schreibe ein kleines Programm, das prüft, ob eine Zahl positiv, negativ oder
null ist.

## Anweisungen

- Erstelle eine Variable `number` und lese ihren Wert mittels `prompt` ein.
- Verwende `if`-Anweisungen, um zu prüfen:
  - Wenn die Zahl größer als 0 ist → gib "The number is positive" aus, ansonsten
    "The number is not positive"
  - Wenn die Zahl kleiner als 0 ist → gib "The number is negative" aus,
    ansonsten "The number is positiv or 0"
  - Wenn die Zahl gleich 0 ist → gib "The number is zero" aus, ansonsten "The
    number is not 0"

**Hinweis: Verwende nur `if`, `<`, `>` und `==`.**

## Abgabe

Bitte gib deine Arbeit in der Datei `main.ts` im Ordner `workspace` ab:
