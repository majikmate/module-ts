# Notenbewertung

Schreibe ein Programm, das eine Punktzahl auswertet und die entsprechende Note
ausgibt.

## Anweisungen

### 1. Variables

- Erstelle eine Variable `score` (Typ: `number`)
- Die Punktzahl ist eine Zahl zwischen 0 und 100

### 2. Logic

Verwende `if` und `else if`-Anweisungen, um die Note zu bestimmen:

- Wenn die Punktzahl 90 oder höher ist → gib "Grade: A" aus
- Sonst, wenn die Punktzahl zwischen 80 und 89 (inklusive) liegt → gib "Grade:
  B" aus
- Sonst, wenn die Punktzahl zwischen 70 und 79 liegt → gib "Grade: C" aus
- Sonst, wenn die Punktzahl zwischen 60 und 69 liegt → gib "Grade: D" aus
- Sonst → gib "Grade: F" aus

## Abgabe

Bitte gib deine Arbeit in der Datei `main.ts` im Ordner `workspace` ab:
