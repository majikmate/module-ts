# Temperatur-Ratgeber

Frage nach der Temperatur und gib aus, was man tun sollte.

## Anweisungen

- Erstelle eine Variable `temperature` für die Temperatur
- Verwende `if`-, `if...else`- oder `if...else if...`-Anweisungen
- Entscheide selbst, bei welcher Temperatur welche Empfehlung gegeben wird
- Prüfe die Temperatur und gib eine entsprechende Empfehlung aus:
  - Drinnen bleiben
  - Warm anziehen
  - Raus gehen
  - Baden gehen
  - In die Kühlbox gehen

## Abgabe

Bitte gib deine Arbeit in der Datei `main.ts` im Ordner `workspace` ab:

[workspace/main.ts](./workspace/main.ts)
