# Getränkebestellung

Frag den User welches Getränk er will.

## Anweisungen

- Erstelle eine Variable `drink` für die Getränkeauswahl
- Verwende `if`-, `if...else`- oder `if...else if...`-Anweisungen
- Biete folgende Getränke zur Auswahl an:
  - Cola
  - Wasser
  - Bier
  - Tee
- Je nach Selektion gib aus: "Danke für deine Bestellung von XY."
  - XY ist dabei das Getränk, das er gewählt hat

## Abgabe

Bitte gib deine Arbeit in der Datei `main.ts` im Ordner `workspace` ab:

[workspace/main.ts](./workspace/main.ts)
