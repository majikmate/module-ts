# Movie Ticket Checker

Schreibe ein Programm, das prüft, ob eine Person einen Film ansehen kann,
basierend auf Alter und verfügbarem Geld.

## Anweisungen

### 1. Variables

Erstelle zwei Variablen:

- `age`: number
- `money`: number (in Euro)

### 2. Logic

Verwende `if-else` und `&&`, um zu prüfen:

- Wenn das Alter ≥ 18 UND Geld ≥ 12 → gib "Ticket purchased" aus
- Sonst, wenn das Alter ≥ 18 UND Geld < 12 → gib "Not enough money" aus
- Sonst → gib "Too young to watch the movie" aus

## Abgabe

Bitte gib deine Arbeit in der Datei `main.ts` im Ordner `workspace` ab:
