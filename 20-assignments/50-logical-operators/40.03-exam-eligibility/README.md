# Exam Eligibility

Schreibe ein Programm, das prüft, ob ein Student für eine fortgeschrittene
Prüfung berechtigt ist.

## Anweisungen

### 1. Variables

Erstelle drei Variablen:

- `grade`: number (0-100)
- `attendance`: number (0-100)
- `extraCredit`: number (0-10)

### 2. Logic

Verwende ein `if-else`-Statement mit sowohl `&&` als auch `||`, um zu prüfen:

- Die Note ist 80 oder höher UND (die Anwesenheit ist 90 oder höher ODER die
  Extrapunkte sind 5 oder höher)

**Ausgabe:**

- Wenn die Bedingungen erfüllt sind → gib "Eligible for advanced exam." aus
- Ansonsten → gib "Not eligible for advanced exam." aus

## Abgabe

Bitte gib deine Arbeit in der Datei `main.ts` im Ordner `workspace` ab:

[workspace/main.ts](./workspace/main.ts)
