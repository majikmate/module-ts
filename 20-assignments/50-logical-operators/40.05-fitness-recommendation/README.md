# Fitness Recommendation

Schreibe ein Programm, das eine Fitnessempfehlung basierend auf Herzfrequenz und
Alter gibt.

## Anweisungen

### 1. Variables

Erstelle zwei Variablen:

- `heartRate`: number (bpm - Schläge pro Minute)
- `age`: number

### 2. Logic

Verwende `if-else` mit `||` und `&&`, um zu prüfen:

- Wenn das Alter < 30 UND die Herzfrequenz < 100 → gib "Light exercise" aus
- Wenn das Alter ≥ 30 UND die Herzfrequenz zwischen 80 und 120 liegt → gib
  "Moderate exercise" aus
- Sonst → gib "Consult a doctor before exercising" aus

Bitte gib deine Arbeit in der Datei `main.ts` im Ordner `workspace` ab:
