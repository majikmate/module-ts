# Speed Limit Warning

Schreibe ein Programm, das prüft, ob ein Fahrer langsamer fahren sollte,
basierend auf Geschwindigkeit und Verkehrslage.

## Anweisungen

### 1. Variables

Erstelle zwei Variablen:

- `speed`: number (in km/h)
- `trafficLevel`: number (von 1-10)

### 2. Logic

Verwende ein `if-else`-Statement mit `||` (logisches ODER), um zu prüfen:

- Die Geschwindigkeit ist über 80 ODER das Verkehrsniveau ist größer als 7

**Ausgabe:**

- Wenn eine der Bedingungen erfüllt ist → gib "Slow down!" aus
- Ansonsten → gib "You are good to go." aus

## Abgabe

Bitte gib deine Arbeit in der Datei `main.ts` im Ordner `workspace` ab:

[workspace/main.ts](./workspace/main.ts)
