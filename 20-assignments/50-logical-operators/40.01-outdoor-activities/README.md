# Temperature Check

Schreibe ein Programm, das prüft, ob heute ein guter Tag für Outdoor-Aktivitäten
ist.

## Anweisungen

### 1. Variables

Erstelle zwei Variablen:

- `temperature`: number (in °C)
- `humidity`: number (in %)

### 2. Logic

Verwende ein `if`-Statement mit `&&` (logisches UND), um zu prüfen:

- Die Temperatur liegt zwischen 20 und 25 UND die Luftfeuchtigkeit ist kleiner
  als 60

**Ausgabe:**

- Wenn beide Bedingungen erfüllt sind → gib "Perfect day for outdoor
  activities!" aus
- Ansonsten → gib "Maybe stay indoors today." aus

## Abgabe

Bitte gib deine Arbeit in der Datei `main.ts` im Ordner `workspace` ab:

[workspace/main.ts](./workspace/main.ts)
