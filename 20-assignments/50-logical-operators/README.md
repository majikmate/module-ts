# Logische Operatoren - Aufgaben

Dieses Kapitel enthält Übungsaufgaben zum Thema logische Operatoren in
TypeScript.

## Übersicht

- [40.01 Outdoor-Aktivitäten](40.01-outdoor-activities/README.md) -
  UND-Verknüpfung verwenden
- [40.02 Geschwindigkeitswarnung](40.02-speed-limit-warning/README.md) -
  ODER-Verknüpfung verwenden
- [40.03 Prüfungsberechtigung](40.03-exam-eligibility/README.md) - Komplexe
  Bedingungen
- [40.04 Kinokarten-Prüfer](40.04-movie-ticket-checker/README.md) - Mehrfache
  Bedingungen
- [40.05 Fitness-Empfehlung](40.05-fitness-recommendation/README.md) -
  Verschachtelte Logik

## Lernziele

Nach Abschluss dieser Aufgaben kannst du:

- Den UND-Operator `&&` verwenden
- Den ODER-Operator `||` verwenden
- Den NICHT-Operator `!` verwenden
- Komplexe logische Ausdrücke schreiben
- Mehrere Bedingungen kombinieren
- Die Operator-Priorität verstehen
- Logische Ausdrücke in if-Anweisungen einsetzen

## Verwendung

Jede Aufgabe enthält:

1. Eine `README.md` Datei mit den Anweisungen
2. Ein `workspace/main.ts` Datei zum Implementieren deiner Lösung
