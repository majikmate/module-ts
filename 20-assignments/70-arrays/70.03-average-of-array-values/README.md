# Average of Array Values

Schreibe ein Programm, das den Durchschnittswert aller Zahlen in einem Array
berechnet.

## Anweisungen

### 1. Aufgabe

- Erstelle ein Array mit mehreren Zahlen
- Berechne den Durchschnitt aller Werte im Array
- Gib den Durchschnitt aus

### 2. Hinweise

- Durchschnitt = Summe aller Elemente ÷ Anzahl der Elemente
- Du kannst eine Schleife verwenden, um die Summe zu berechnen
- Die Anzahl der Elemente erhältst du mit `array.length`

### 3. Beispiel

```TypeScript
const numbers = [10, 20, 30, 40, 50];
```

Erwartete Ausgabe:

```
Durchschnitt: 30
```

(Berechnung: (10 + 20 + 30 + 40 + 50) / 5 = 150 / 5 = 30)
