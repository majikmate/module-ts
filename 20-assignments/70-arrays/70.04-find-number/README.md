# Find Largest Number

Schreibe ein Programm, das den größten Wert in einem Array findet, ohne
`Math.max()` zu verwenden.

## Anweisungen

### 1. Aufgabe

- Erstelle ein Array mit mehreren Zahlen
- Finde den größten Wert im Array durch manuelles Vergleichen
- Gib den größten Wert aus
- **Nicht erlaubt:** `Math.max()` verwenden

### 2. Hinweise

- Initialisiere eine Variable mit dem ersten Element des Arrays als Startwert
  für den größten Wert
- Durchlaufe das Array mit einer Schleife
- Vergleiche jedes Element mit dem aktuell größten Wert
- Wenn ein Element größer ist, aktualisiere den größten Wert

### 3. Beispiel

```TypeScript
const numbers = [12, 45, 23, 67, 34, 89, 56];
```

Erwartete Ausgabe:

```
Größter Wert: 89
```
