# Array-Operationen

Erstelle ein Array mit folgenden Werten: 4, 1, 2, 3 und füge die Werte 17 und
199 dazu.

## Anweisungen

### 1. Aufgabe

- Erstelle ein Array mit den Werten 4, 1, 2, 3
- Füge die Werte 17 und 199 zum Array hinzu

### 2. Teilaufgaben

#### a. Gib alle Werte in einer Schleife aus!

- Verwende eine `for`-Schleife, um durch das Array zu iterieren
- Gib jeden Wert mit `console.log()` aus

#### b. Zähle alle Werte zusammen und gib das Ergebnis aus

- Erstelle eine Variable für die Summe (z.B. `summe`) und initialisiere sie mit
  0
- Durchlaufe das Array mit einer Schleife
- Addiere jeden Wert zur Summe
- Gib die Gesamtsumme aus

#### c. Berechne den Mittelwert der Zahlen und gib das Ergebnis aus

- Verwende die berechnete Summe aus Teilaufgabe b
- Teile die Summe durch die Anzahl der Elemente im Array
- Du kannst die Anzahl der Elemente mit `array.length` ermitteln
- Gib den Mittelwert aus

### 3. Hinweise

- Verwende `push()` um Elemente zu einem Array hinzuzufügen
- Die Array-Länge erhältst du mit `array.length`
- Der Mittelwert = Summe aller Werte / Anzahl der Werte

### 4. Erwartete Ausgabe

```
4
1
2
3
17
199
226
37.666666666666664
```
