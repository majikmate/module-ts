# Arrays - Aufgaben

Dieses Kapitel enthält Übungsaufgaben zum Thema Arrays in TypeScript.

## Übersicht

- [70.01 Werte](70.01-values/README.md) - Arbeiten mit Array-Werten
- [70.02 Namen](70.02-names/README.md) - String-Arrays verwenden
- [70.03 Durchschnitt von Array-Werten](70.03-average-of-array-values/README.md) -
  Durchschnittsberechnung
- [70.04 Zahl finden](70.04-find-number/README.md) - Elemente in Arrays suchen
- [70.05 Summe aller Elemente](70.05-sum-of-all-elements/README.md) -
  Array-Summen berechnen

## Lernziele

Nach Abschluss dieser Aufgaben kannst du:

- Arrays erstellen und mit Werten füllen
- Auf Array-Elemente über Indizes zugreifen
- Arrays mit `for`-Schleifen durchlaufen
- Arrays mit `for...of`-Schleifen durchlaufen
- Berechnungen mit Array-Elementen durchführen
- Die `.length` Eigenschaft verwenden

## Verwendung

Jede Aufgabe enthält:

1. Eine `README.md` Datei mit den Anweisungen
2. Ein `workspace/main.ts` Datei zum Implementieren deiner Lösung
