# Freunde-Liste mit Array

Erstelle ein Array mit folgenden Strings: "Susi", "Paula", "Hans" und erweitere
es dynamisch.

## Anweisungen

### 1. Aufgabe

- Erstelle ein Array mit den Strings "Susi", "Paula", "Hans"
- Gib folgenden Satz aus: "Meine Freunde sind Susi, Paula und Hans"
- Füge "Sepp" zum Array hinzu
- Gib nun folgenden Satz aus: "Meine Freunde sind Susi, Paula, Hans und Sepp"

### 2. Hinweise

- Verwende `push()` um ein Element zum Array hinzuzufügen
- Du kannst die Array-Elemente mit dem Index ansprechen: `array[0]`, `array[1]`,
  etc.
- Verwende `console.log()` mit String-Konkatenation (mit `+`) um die Ausgabe zu
  erzeugen
- Achte auf die korrekte Satzstellung mit Kommas und "und" vor dem letzten Namen

### 3. Erwartete Ausgabe

```
Meine Freunde sind Susi, Paula und Hans
Meine Freunde sind Susi, Paula, Hans und Sepp
```

## Abgabe

Bitte gib deine Arbeit in der Datei `main.ts` im Ordner `workspace` ab:

[workspace/main.ts](./workspace/main.ts)
