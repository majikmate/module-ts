# Berechnungen

## Angaben

- Erstelle eine Variable `bananaPricePerKilo` mit dem Wert `2.29`
- Erstelle eine Variable `applePricePerKilo` mit dem Wert `3.49`

- Ein Apfel hat durchschnittlich ein Gewicht von 0.34 Kilo
- Eine Banane hat durchschnittlich ein Gewicht von 0.22 Kilo

## Ausgabe

Gib folgende Texte auf der Konsole aus:

- Ein Kilogramm Bananen entspricht einer durchschnittlichen Anzahl von xx
  Bananen
- Ein Kilogramm Äpfel entspricht einer durchschnittlichen Anzahl von xx Äpfeln

- 17 Äpfel kosten xx €
- 5 Bananen kosten xx €

- 5 Tonnen Äpfel kosten xx €
- 5 Tonnen Bananen kosten xx €

Ersetze xx in obigen Texten mit den berechneten Werten.
