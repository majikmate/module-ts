# Variablen & Operationen

Texte werde in TypeScript als Zeichenketten, auch Strings genannt, gespeichert.

Gib die Zeichenketten `Hallo` und `Tolle Sache` auf der Konsole aus:

```TypeScript
console.log("Hallo");
console.log("Tolle Sache");
```

Nun arbeite mit den Variablen `a`, `b`, und `c`:

```TypeScript
// Variablen
let a = 7;
let b = 8;
let c = 10;
```

Berechne folgende Regebnisse und gib diese auf der Konsole aus:

```TypeScript
// a + b + c
// (a + b) / c
// c -a + b
// a + b + c * 3
// (a + b + c)/4
```
